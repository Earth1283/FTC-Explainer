Robot Code
==========

This section documents team **32008**'s FTC robot controller code, located in
``src/`` at the root of this repository. The code targets the FTC SDK
(``org.firstinspires.ftc.teamcode``) and uses `FTCLib
<https://docs.ftclib.org/ftclib/>`_ for command-based structuring and
`Pedro Pathing <https://pedropathing.com/>`_ for autonomous path following.
